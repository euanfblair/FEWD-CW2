# Family Organiser

A family event management application.

## Prerequisites

- Node.js (v16+)
- npm

## Installation

```bash
# Install server dependencies
cd server
npm install

# Install client dependencies
cd ../client
npm install
```

## Running the Application

**Terminal 1 - Start the server:**
```bash
cd server
npm start
```

**Terminal 2 - Start the client:**
```bash
cd client
npm run dev
```

Open http://localhost:5173 in your browser.
